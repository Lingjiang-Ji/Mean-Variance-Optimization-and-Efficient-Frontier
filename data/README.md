# Data Description

This directory contains example return data used for mean–variance
portfolio optimization and efficient frontier analysis.

## Time Period

- Sample period: **1960–2002**
- Frequency: annual total returns

## Asset Classes

The dataset consists of total returns from the following four asset classes:

1. S&P 500 equity index
2. Long-term bonds
3. Money market account
4. NASDAQ stocks

## Notes

- The dataset is used to estimate expected returns and the covariance matrix
  for portfolio optimization.
- The optimization code does not depend on this specific dataset and can be
  applied to other return series with the same format.
